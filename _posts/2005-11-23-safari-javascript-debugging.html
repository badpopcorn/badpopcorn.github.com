I really really hate Javascript. And out of all the things that I can whine on about, the fact that it's hard to debug nearly tops them all.

So, it makes me happy to have discovered that Safari has a hidden Javascript debug console. Just type the following in a terminal and restart Safari:

{% highlight bash %} defaults write com.apple.Safari IncludeDebugMenu 1 {% endhighlight %}

(I am running Safari 2.0.2 (416.12) on OS X 10.4.)
